## Описание
Desktop-приложение для планирования и учёта тренировок. Позволяет добавлять, удалять и фильтровать тренировки по дате и типу. Данные сохраняются в JSON-файл.

Технологии:
*    Python 3
*   `tkinter`: Для создания GUI.
*   `json`: Для сохранения и загрузки данных.
*   `datetime`: Для работы с файловой системой и датой/временем.



## Функциональность
- Добавление тренировок с указанием даты, типа и длительности
- Валидация ввода (формат даты, положительная длительность)
- Фильтрация тренировок по типу и дате
- Удаление отдельных тренировок или всех записей
- Автоматическое сохранение в JSON
- Сортировка по дате

## Установка и запуск

### Требования
- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, datetime)

### Запуск
```bash
python training_planner.py
Использование
Добавление тренировки
Введите дату в формате ГГГГ-ММ-ДД(например, 2024-12-25)

Выберите тип тренировки из выпадающего списка

Укажите длительность в минутах (положительное целое число)

Нажмите "Добавить тренировку"

Фильтрация
По типу: выберите тип тренировки из фильтра

По дате: введите конкретную дату

Нажмите "Применить фильтр"

"Сбросить фильтр" - показывает все тренировки

Удаление
Выберите тренировку в таблице и нажмите "Удалить выбранную"

"Очистить все" - удаляет все тренировки (с подтверждением)

Примеры использования
Пример 1: Ежедневный учёт
2024-12-20, Бег, 30 минут

2024-12-21, Плавание, 45 минут

2024-12-22, Силовая, 60 минут

Пример 2: Фильтрация
Добавить 5 тренировок разных типов

Выбрать фильтр "Бег" → покажет только беговые тренировки

Ввести дату "2024-12-20" → покажет тренировки только за этот день

Тесты (примеры проверки)
Тест 1: Валидация даты
Попытка ввода 32.13.2024 → ошибка

Ввод 2024-12-25 → успешно

Тест 2: Валидация длительности
Ввод -10 → ошибка

Ввод 0 → ошибка

Ввод 45 → успешно

Тест 3: Фильтрация
Добавить: Бег (25.12.2024), Плавание (26.12.2024)

Фильтр по типу "Бег" → 1 запись

Фильтр по дате "2024-12-25" → 1 запись

Тест 4: Сохранение данных
Добавить тренировку → закрыть приложение

Открыть заново → тренировка сохранилась
